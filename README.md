# \# 🛒 Cartify - ASP.NET E-commerce Web Application

# 

# Cartify is a full-featured e-commerce web application built with ASP.NET MVC. It allows users to browse products, manage carts, place orders, and for admins to manage the platform efficiently.

# 

# \## 📌 Features

# 

# \### 🔍 Customer Features

# 

# \* Browse and search for products

# \* Add/remove items from cart

# \* User registration \& login (authentication)

# \* View order history and profile

# 

# \### 🛠️ Admin Features

# 

# \* Product management (CRUD)

# \* Category management

# \* Order management

# \* Customer management

# \* Dashboard with sales insights

# 

# \### 💡 Other Highlights

# 

# \* Responsive design (Bootstrap)

# \* Session-based cart

# \* Authentication \& Authorization

# \* Secure password hashing

# \* Validation using Data Annotations

#

# \## 🛠️ Tech Stack

# 

# | Layer          | Technology            |

# | -------------- | --------------------- |

# | Frontend       | HTML, CSS, Bootstrap  |

# | Backend        | ASP.NET MVC (C#)      |

# | Database       | SQL Server (EF Core)  |

# \## 🚀 Getting Started

# 

# \### Prerequisites

# 

# \* \[.NET SDK](https://dotnet.microsoft.com/en-us/download)

# \* SQL Server / LocalDB

# \* Visual Studio 2022 or later

# 

# \### Installation

# 

# 1\. Clone the repository:

# 

# &nbsp;  ```bash

# &nbsp;  git clone https://github.com/yourusername/Cartify.git

# &nbsp;  cd Cartify

# &nbsp;  ```

# 

# 2\. Set the connection string in `appsettings.json`:

# 

# &nbsp;  ```json

# &nbsp;  "ConnectionStrings": {

# &nbsp;    "DefaultConnection": "Server=.;Database=CartifyDB;Trusted\_Connection=True;"

# &nbsp;  }

# &nbsp;  ```

# 

# 3\. Apply migrations and seed the database:

# 

# &nbsp;  ```bash

# &nbsp;  dotnet ef database update

# &nbsp;  ```

# 

# 4\. Run the application:

# 

# &nbsp;  ```bash

# &nbsp;  dotnet run

# &nbsp;  ```
