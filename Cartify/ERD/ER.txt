User:
   - UserID (PK)
   - Username (unique)
   - Email (unique)
   - Password
   - FName
   - LName
   - Address
   - PNumber
   - RegistrationDate
   - LastLoginDate
   - IsActive
   - Address
   - Type 

Order:
   - OrderID (PK)
   - OrderDate
   - OrderStatus
   - TaxAmount
   - ShippingCost
   - TotalAmount
   - TrackingNumber
   - ShippingMethod

OrderItem:
   - OrderItemID (PK)
   - Quantity
   - Price
   - Discount

Payment:
   - PaymentID (PK)
   - PaymentMethod
   - Amount
   - PaymentDate

Cart:
    - CartID (PK)
    - UserID (FK)
    - CreatedDate
    - LastUpdated

CartItem:
    - CartItemID (PK)
    - Quantity
    - DateAdded

Product:
   - ProductID (PK)
   - Name
   - Description
   - Price
   - DiscountedPrice 
   - StockQuantity
   - DateAdded
   - LastUpdated
   - IsActive

Category:
   - CategoryID (PK)
   - Name
   - Description
   - Reviews
   - ParentCategoryID (FK - self-reference)


## ERD Relationships Summary:

- Users can have multiple Addresses (1-to-many)
- Users can have multiple Orders (1-to-many)
- Users can make multiple Payments (1-to-many)
- Payment can have multiple Payment Methods (1-to-many)
- Product added to Shopping cart (1-to-many)
- Products belong to Categories (many-to-many via junction table)
- Orders contain multiple OrderItems (1-to-many)
- Products can have multiple Reviews (1-to-many)
- ShoppingCart contains multiple CartItems (1-to-many)
